Meu Repositório Kodi

Bem-vindo ao meu repositório oficial. Aqui você encontra meus addons e atualizações automáticas.

Baixar ZIP do Repositório

Como Instalar:

  1. Baixe o arquivo ZIP acima.
  2. Abra o Kodi e vá em Add-ons.
  3. Clique no ícone da Caixa Aberta (Instalador de pacotes).
  4. Escolha Instalar via arquivo ZIP.
  5. Após instalado, escolha Instalar via repositório para encontrar meus addons.
